# Uderia Tutorial Series

**26 videos · Two tracks · Recorded in Kapwing**

The series is organized around two principles that run through every segment:

- **IFOC Methodology** — Ideate · Focus · Optimize · Coordinate — the four execution modes that match any problem to the right intelligence
- **Design Principles** — nine "From → To" promises introduced in the intro, delivered by every tutorial that follows

---

## Two Tracks

| Track | Who it's for | Start here |
|---|---|---|
| **Business** | Analysts, managers, non-technical users | [00_introduction →](#tier-00-introduction) then [01_jumpstart/business →](#business-track) |
| **Technical** | DBAs, engineers, developers, admins | [00_introduction →](#tier-00-introduction) then [01_jumpstart/technical →](#technical-track) |

Both tracks share the Introduction and Core Features tiers. User Journeys are relevant to both.
